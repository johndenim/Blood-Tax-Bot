# Blood Tax Bot

A psychological number-based duel where survival depends on how you manage what you spend.

## What is this?

Blood Tax is a Discord game where two players secretly choose numbers each round.

- Go too low → you fail the tax  
- Go too high → you get punished for greed  
- Play it right → you survive  

The goal isn’t to win big.  
It’s to **outlast your opponent**.

---

## How it works

- Each player has numbers from 1–10  
- Each number can only be used once  
- Every round has a target value (“Tax”)  
- Players secretly submit numbers  

Outcomes:
- Meet the tax → safe  
- Too low → lower number loses HP  
- Too high → higher number loses HP  

---

## Features

- Hidden bidding system  
- Finite resource strategy (numbers run out)  
- Psychological gameplay (reading your opponent matters more than math)  
- Simple rules, high tension
